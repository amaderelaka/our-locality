Drop scanned PDFs of school magazines and historic documents here.

Suggested conventions:
- Lowercase filenames with hyphens (e.g. khirpai-school-magazine-1987.pdf).
- Keep file sizes reasonable (under 5 MB per file) so the site loads quickly.
  Use free tools like ilovepdf.com or Smallpdf to compress large scans.
- Always note permissions and credit on the Writings page entry.
