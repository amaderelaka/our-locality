Drop your photographs into this folder.

Suggested conventions:
- Lowercase filenames, hyphens instead of spaces (e.g. rash-mela-2024.jpg).
- Keep file sizes reasonable (under 500 KB) so the site loads quickly. Free tools like Squoosh (squoosh.app) can compress images in your browser.
- Add a credit and date to every photo's caption on the gallery page.
